# Cloud Pika — UI 设计功能清单 & Figma 设计提示词

> 本文档面向专业 UI 设计师，完整描述 Cloud Pika 桌面端云存储管理客户端的所有页面、组件、交互状态，作为 Figma 设计的输入规范。

---

## 一、产品概述

Cloud Pika 是一款跨平台（macOS / Windows / Linux）桌面云存储管理客户端，使用 Tauri 2 构建，支持 9 家云存储服务商的统一管理。定位是**精致、专业、高效**的文件管理工具，面向开发者和运维人员。

### 支持的云厂商（需准备 Logo/Icon）

| 序号 | 厂商 | 英文标识 | 特有功能 |
|------|------|---------|---------|
| 1 | 七牛云 | Qiniu | CDN 刷新、URL 抓取上传、域名管理 |
| 2 | 腾讯云 COS | Tencent COS | — |
| 3 | 阿里云 OSS | Alibaba OSS | — |
| 4 | AWS S3 | AWS S3 | — |
| 5 | 青云 QingStor | QingStor | — |
| 6 | 京东云 | JD Cloud | — |
| 7 | 又拍云 | UPYun | — |
| 8 | MinIO | MinIO | 自建存储 |
| 9 | 金山云 KS3 | Ksyun KS3 | — |

### 技术约束（影响设计）

- 窗口尺寸：默认 1280×820，最小 800×600，需适配到 1920×1080+
- 自定义标题栏（无系统原生标题栏）
- 支持深色/浅色两套主题
- 支持中文/英文两种语言
- 文件数量可达数千乃至数万，需考虑虚拟滚动场景
- 桌面端操作习惯：右键菜单、键盘快捷键、拖拽

## 二、全局框架 & 布局结构

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  自定义标题栏 (Title Bar)                                │
├────────┬────────────────────────────────────────────────┤
│        │  工具栏 (Toolbar)                               │
│  侧边栏 ├────────────────────────────────────────────────┤
│ Sidebar │                                                │
│        │  主内容区 (Main Content)                         │
│        │                                                │
│        │                                                │
│        │                                                │
├────────┴────────────────────────────────────────────────┤
│  状态栏 (Status Bar)                                     │
└─────────────────────────────────────────────────────────┘
```

### 2.2 自定义标题栏 (Title Bar)

**内容元素：**
- 左侧：窗口控制按钮区（macOS: 红黄绿交通灯，Windows: 最小化/最大化/关闭）
- 左中：应用 Logo + 名称 "Cloud Pika"
- 中间：可选的全局搜索入口 / 命令面板触发器（⌘K / Ctrl+K）
- 右侧：语言切换（中/EN）、主题切换（☀/🌙）、通知/传输入口、用户头像/账户

**状态：** 正常、窗口获焦、窗口失焦（颜色变淡）

### 2.3 侧边栏 (Sidebar)

**宽度：** 默认 240px，可折叠为 48px（仅图标模式），可拖拽调整宽度

**内容结构（自上而下）：**

1. **快速导航区**
   - 所有文件（总览）
   - 收藏夹 / 星标存储桶
   - 最近访问

2. **云账户区**（可折叠分组）
   - 每个账户：厂商 Icon + 账户别名 + 连接状态指示（绿点=在线）
   - 展开后：该账户下的 Bucket 列表
   - 每个 Bucket：存储桶 Icon + Bucket 名称
   - Hover 时显示更多操作按钮（⋯）

3. **底部区域**
   - 设置入口
   - 添加账户按钮
   - 折叠/展开按钮

**状态：** 正常、Hover、选中(Active)、折叠模式、展开模式

### 2.4 状态栏 (Status Bar)

**内容：** 当前连接信息（厂商+Bucket名） | 对象数量 "42 个对象, 3 个文件夹" | 已用存储 "1.2 GB" | 传输状态快捷入口 "↑2 ↓1"

## 三、页面清单 & 功能详情

### 页面 1：启动加载页 (Bootstrap Screen)

**场景：** 应用启动时，后端 Sidecar 服务初始化期间展示

**内容：**
- 应用 Logo（居中、较大）
- 应用名称 "Cloud Pika"
- 进度指示器（进度条或步骤文字：正在启动服务… → 正在连接… → 准备就绪）
- 版本号（底部小字）

**状态：**
- 加载中（默认）
- 加载失败：显示错误信息 + "重试"按钮 + "查看日志"链接

---

### 页面 2：登录 / 账户管理页 (Login Page)

**场景：** 首次启动无账户时 / 从侧边栏点击"添加账户" / 管理已有账户

**布局：** 左右分栏或上下分区

**左侧/上方 — 已保存账户列表：**
- 账户卡片：厂商 Logo + 别名 + 最后使用时间 + 右上角 ⋯ 菜单（编辑/删除）
- 空状态：无已保存账户时的引导文字
- 点击账户卡片直接登录进入文件浏览器

**右侧/下方 — 添加/编辑账户表单：**

| 表单字段 | 控件 | 备注 |
|---------|------|------|
| 云厂商 | 卡片式选择器（9个厂商，带 Logo + 名称） | 不是下拉框，是可视化选择 |
| 别名 | 文本输入框 | 占位符："给账户起个名字" |
| AccessKey | 文本输入框 | — |
| SecretKey | 密码输入框 + 显示/隐藏切换 | Eye/EyeOff 图标 |
| ▸ 高级选项（折叠） | — | 点击展开 |
| └ 区域/Region | 下拉选择 | 按厂商动态变化 |
| └ 自定义 Endpoint | 文本输入框 | MinIO 等自建存储必填 |
| └ 服务名称 | 文本输入框 | 部分厂商需要 |
| 验证并保存 | 主按钮 | 带 loading 状态和结果反馈 |

**状态：**
- 表单空白（初始）
- 填写中（字段校验提示）
- 验证中（按钮 loading）
- 验证成功（绿色勾 + 自动跳转）
- 验证失败（红色错误提示：网络错误 / 凭证无效 / 区域不匹配）

### 页面 3：文件浏览器 (Bucket Page) — 核心页面

**场景：** 用户登录后的主要工作区，占 90% 使用时间

#### 3.1 工具栏 (Toolbar)

**从左到右排列：**

| 元素 | 类型 | 说明 |
|------|------|------|
| 面包屑导航 | 可点击路径 | / > bucket-name > images > 2024，每段可点击跳转 |
| — 弹性间距 — | | |
| 搜索框 | 输入框 | 带搜索图标、清除按钮(X)、300ms 防抖 |
| 视图切换 | 图标按钮组 | 列表 / 网格，二选一高亮 |
| 排序 | 下拉菜单 | 按名称 / 大小 / 修改时间，升序/降序 |
| 新建文件夹 | 按钮 | 图标+文字 或 纯图标 |
| 上传 | 主操作按钮(Primary) | 图标+文字 "上传"，下拉子菜单：上传文件 / 上传文件夹 / URL抓取上传(七牛) / 剪贴板上传 |
| 更多 | ⋯ 按钮 | CDN 刷新(七牛) / 批量导出URL / 刷新列表 |

#### 3.2 列表视图 (Table View)

**表格列：**

| 列 | 宽度 | 内容 | 交互 |
|----|------|------|------|
| ☐ | 32px | 复选框（全选/单选） | 点击勾选，Shift+点击范围选 |
| 图标 | 32px | 文件类型图标（文件夹/图片/文档/视频/音频/代码/压缩包/通用） | — |
| 名称 | 自适应 | 文件名，文件夹加粗 | 单击选中，双击进入/预览 |
| 大小 | 100px | 人类可读格式 "1.2 MB" | 点击列头排序 |
| 修改时间 | 160px | 相对时间 "3分钟前" / 绝对时间 "2024-01-15 14:30" | 点击列头排序 |
| 操作 | 40px | ⋯ 下拉菜单 | 详见右键菜单 |

**行状态：** 正常、Hover（背景变色）、选中（强调色背景）、拖拽中

**列头状态：** 正常、Hover、排序中（显示 ▲/▼ 箭头）

**空状态：** 桶内无文件时展示"空空如也"引导 + "上传文件"按钮

#### 3.3 网格视图 (Grid View)

**卡片内容：**
- 缩略图区域：图片文件加载缩略图，其他文件显示大号文件类型图标
- 文件名：单行截断，Tooltip 显示全名
- 文件大小（小字灰色）

**卡片状态：** 正常、Hover（轻微上浮+阴影/边框变化）、选中（强调色边框/勾选角标）

**网格列数：** 根据容器宽度自适应（4~8列）

#### 3.4 右键菜单 (Context Menu)

**右键文件/文件夹：**

```
打开 / 进入文件夹
预览                     (仅图片/视频/文本)
──────────────────
复制链接
复制为 Markdown
复制为 HTML
──────────────────
下载
重命名
移动到...               (如支持)
──────────────────
删除                     ← 红色文字
```

**右键空白区域：**

```
上传文件
上传文件夹
新建文件夹
──────────────────
粘贴（剪贴板图片上传）
──────────────────
刷新
全选
```

#### 3.5 批量选择操作栏 (Selection Bar)

**触发：** 当选中 ≥1 个文件时，从底部浮出

**内容：** "已选 3 个文件" | [下载] [复制链接] [批量导出URL] [删除(红色)] | [取消选择 ✕]

**动画：** 从底部滑入/滑出

#### 3.6 拖拽上传覆盖层 (Drag Overlay)

**触发：** 文件从系统拖入应用窗口时

**内容：** 半透明遮罩 + 居中大图标 + "松开以上传文件到 {bucket}/{path}" + 虚线边框

**状态：** 拖入时显示、拖出时消失、松开后触发上传

### 页面 4：传输管理 (Transfers)

**入口：** 标题栏/状态栏的传输图标，或侧边栏导航

**可以是独立页面，也可以是从右侧滑出的面板（Panel），两种都需要设计**

#### 4.1 顶部统计栏

```
上传中 2  ·  下载中 1  ·  排队 5  ·  已完成 23  ·  失败 1
```

每个状态数字用对应颜色的 Badge

#### 4.2 传输任务卡片

```
┌──────────────────────────────────────────────┐
│ ↑ sunset.jpg                                 │
│   my-bucket / images / 2024 /                │
│   上传中 · 2.4 MB/s · 剩余 12s               │
│   ████████████████░░░░░░░  72%       [暂停]  │
└──────────────────────────────────────────────┘
```

**每张卡片包含：**
- 方向图标（↑上传 / ↓下载）
- 文件名
- 目标路径（小字灰色）
- 状态文本 + 传输速度 + 剩余时间
- 进度条 + 百分比
- 操作按钮：暂停/继续、取消(排队/进行中) / 重试(失败) / 清除(已完成)

**卡片状态颜色：**

| 状态 | 颜色 | 操作 |
|------|------|------|
| 排队中 queued | 灰色 | 取消 |
| 传输中 running | 蓝色/主题色 | 暂停、取消 |
| 已暂停 paused | 黄色 | 继续、取消 |
| 已完成 completed | 绿色 | 清除 |
| 失败 failed | 红色 | 重试、清除 |

#### 4.3 全局操作

- 暂停全部 / 继续全部
- 清除已完成
- 清除全部

### 页面 5：设置页 (Settings)

**布局：** 左侧 Tab 导航 或 单页分组滚动

#### 分组结构：

**外观 (Appearance)**

| 设置项 | 控件 | 描述 |
|--------|------|------|
| 主题模式 | 下拉：浅色/深色/跟随系统 | 界面配色方案 |
| 语言 | 下拉：中文/English | 界面显示语言 |

**存储 (Storage)**

| 设置项 | 控件 | 描述 |
|--------|------|------|
| HTTPS 优先 | 开关 Toggle | 生成文件链接时优先使用 HTTPS |
| 默认复制格式 | 下拉：URL / Markdown / HTML | 复制链接的默认格式 |
| 默认下载路径 | 路径选择（带浏览按钮） | 文件下载的默认保存位置 |
| 分页加载 | 开关 Toggle | 大量文件时分页加载 |

**传输 (Transfer)**

| 设置项 | 控件 | 描述 |
|--------|------|------|
| 并发上传数 | 数字输入 (1~10) | 同时上传的最大文件数 |
| 大文件分片阈值 | 下拉：50MB/100MB/200MB | 超过阈值自动分片 |
| 上传成功后自动复制链接 | 开关 Toggle | — |

**安全 (Safety)**

| 设置项 | 控件 | 描述 |
|--------|------|------|
| 隐藏删除按钮 | 开关 Toggle | 隐藏文件列表中的删除操作，防止误删 |
| 删除前确认 | 开关 Toggle（默认开） | 删除文件前弹出确认框 |

**关于 (About)**

| 内容 | 说明 |
|------|------|
| 应用 Logo + 名称 | — |
| 版本号 | v0.1.0 |
| 开源地址 | 可点击链接 |
| 检查更新 | 按钮 |
| 许可证 | MIT |

### 页面 6：文件预览 (Preview)

**触发：** 双击图片/视频/文本文件，或右键菜单选择"预览"

**类型 A — 图片预览：**
- 全屏模态遮罩，图片居中展示
- 缩放控制（+/-/适应/原尺寸）、旋转
- 底部信息栏：文件名 | 尺寸 | 大小 | 修改时间
- 操作按钮：复制链接、下载、删除
- 左右箭头切换同目录下一张/上一张图片
- ESC 或点击遮罩关闭

**类型 B — 视频预览：**
- 通过签名 URL 的视频播放器
- 播放/暂停、进度条、音量、全屏

**类型 C — 文本/代码预览：**
- 等宽字体代码查看器
- 语法高亮（如支持）
- 行号显示

### 页面 7：命令面板 (Command Palette)

**触发：** ⌘K (Mac) / Ctrl+K (Windows)

**布局：** 居中浮层，类似 Spotlight / Raycast

**内容区域：**
- 顶部搜索输入框（自动聚焦）
- 分组结果列表（键盘上下选择，Enter 执行）：
  - **最近文件** — 最近打开/上传的文件
  - **操作** — 上传文件、新建文件夹、刷新、打开设置…
  - **导航** — 切换账户、切换 Bucket
- 每项右侧显示快捷键提示（如有）

**状态：** 打开动画（从上滑入+Fade）、关闭（ESC/点击外部）
